🎯 CASOS DE USO
═══════════════════════════════════════════════════════════════════════════

1. EXTRAÇÃO SIMPLES
   const extractor = new CNPJExtractor();
   const result = await extractor.extract('https://empresa.com.br');
   → { cnpj: "11.222.333/0001-81", confidence: 0.95, ... }

2. PROCESSAMENTO EM LOTE
   for (const url of urls) {
     const result = await extractor.extract(url);
   }
   → Processa múltiplas URLs sequencialmente

3. API REST
   POST /api/extract-cnpj
   → Endpoint pronto para produção

4. VALIDAÇÃO LOCAL
   extractor.validateCNPJ('11.222.333/0001-81')
   → true/false
